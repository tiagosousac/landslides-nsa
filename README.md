# landslides-nsa

## Documentation

* [Nasa's global landslide catalog](https://maps.nccs.nasa.gov/arcgis/apps/MapAndAppGallery/index.html?appid=574f26408683485799d02e857e5d9521)
This was our used database for the global landslides.

* [Adobe XD prototype](https://xd.adobe.com/view/adcc94aa-f1bd-4063-9879-e52b1310ca7f-f89f/screen/aba6f545-0d78-4e20-b4e8-b1c6589b095f)
This is the prototype that we based our application. Not all features were able to be implemented during the weekend.

## How to run

* After downloading the repository, make sure you have Python 3.8+ installed in your machine.

``` 
source backend/venv/bin/activate 
cd backend
flask run
```

By running this command, you'll have our project running in your machine.

You can access the URL displayed in the command line for testing.
